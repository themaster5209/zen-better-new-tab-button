# Better New Tab button

![Better New Tab button](https://github.com/user-attachments/assets/491f3bac-897c-4e9b-bc76-51f802fa8727)

### Features
- Removed "New Tab" text and centered plus icon for a clean look
- Fine-tuned animations, such as the plus icon rotating on click
- Preferences to adjust border radius of various components in the sidebar

### Installation
Using Sine is the optimal way as of v1.0.3 as there is now support for preferences. Simply copy the URL to this repository and paste it into the box directly under the marketplace! Alternatively, you can search for this mod's name in Sine's marketplace.

### Additional information
- If you have any feature requests, feel free to submit PRs or contact me on Discord: themaster_5209
- [Nebula](https://github.com/JustAdumbPrsn/Zen-Nebula/tree/main) on macOS is the only theme that I have tested on, try/use with other themes at your own discretion
- Recommended to have New Tab button placed at the top (can be changed in 'about:preferences#zenLooks')
- Border radius preference for tabs unfortunately doesn't support Essential tabs, so those will stay at Zen's default value
